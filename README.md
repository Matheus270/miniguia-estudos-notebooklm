# Projeto NotebookLM - Bootcamp DIO
## Lupo - Primeiros Passos com Inteligência Artificial

### 1. Objetivo:

  Esse é um projeto de estudos para Pentest e Segurança da Informação usando o Google NotebookLM, focando em aprender conceitos e utilizações da ferramenta Wireshark.  
  
  O objetivo é descrever funções modo de uso e aplicações práticas para a ferramenta, fazendo com que a IA (notebookLM) apresente cada passo do processo para que tudo funcione, desde os primeiros comandos até a utilização dos dados obtidos nas próximas etapas do pentest.
